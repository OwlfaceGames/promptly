## Installation

One-line install:
```bash
curl -sSL https://raw.githubusercontent.com/owlfacegames/promptly/master/install.sh | bash
```

Then run:
```bash
promptly
```

Select your theme and restart your terminal.

## Requirements
- Zsh shell
- For icon theme: Nerd Font installed in terminal
